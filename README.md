# MAVIS GROUP — КП Генератор

## Деплой на GitHub Pages (5 минут)

### Шаг 1: Создать репозиторий
1. Зайдите на https://github.com/new
2. Название: `mavis-kp` (или любое)
3. Поставьте **Public**
4. Нажмите **Create repository**

### Шаг 2: Загрузить файлы
1. На странице репозитория нажмите **uploading an existing file**
2. Перетащите ВСЕ 6 файлов из этого архива:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
   - `icon-192.png`
   - `icon-512.png`
3. Нажмите **Commit changes**

### Шаг 3: Включить Pages
1. Зайдите в **Settings** → **Pages** (в левом меню)
2. Source: **Deploy from a branch**
3. Branch: **main** → папка **/ (root)**
4. Нажмите **Save**

### Шаг 4: Подождите 1-2 минуты
Сайт будет доступен по адресу:
```
https://ВАШ-ЛОГИН.github.io/mavis-kp/
```

### Шаг 5: Установить на рабочий стол
1. Откройте ссылку в **Google Chrome**
2. В адресной строке появится иконка ⊕ **«Установить»**
3. Нажмите — приложение появится на рабочем столе

## Файлы
- `index.html` — приложение
- `manifest.json` — настройки PWA
- `sw.js` — сервис-воркер (кэш для офлайн)
- `icon.svg` / `icon-192.png` / `icon-512.png` — иконки
